# Canary Shell

## What is it?

Canary shell is a shell made in ruby to replace bash or powershell.

## Installing

### Linux

Debian/Ubuntu:
1. Install ruby
`sudo apt-get install ruby ruby-devel`

2. Clone the repo
`git clone https://github.com/gitipedras/canary-shell.git`

3. Go to the directory
`cd canary-shell`

4. Execute ruby
`ruby main.rb`
